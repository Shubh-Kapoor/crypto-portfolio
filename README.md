# 🚀 Crypto Portfolio

A modern cryptocurrency portfolio tracking application with real-time market data and AI-powered sentiment analysis.

## ✨ Features
- Live cryptocurrency prices and market data
- Interactive price charts
- AI Sentiment Analysis powered by Claude API
- Support for INR, USD, and EUR currencies
- Exchange listings

## 🛠️ Tech Stack
- React.js
- Chakra UI
- Tailwind CSS
- Chart.js
- CoinGecko API
- Claude API (Anthropic)
- Vite

## 💻 Installation
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
